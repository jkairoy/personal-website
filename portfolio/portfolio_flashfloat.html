THE PROJECT
FlashFloat is a little CLI that lets you upload images to your AWS S3 bucket. It will then document the images and the links that point to them so you can use these images in web development in any context you choose. You can use flashfloat to update these images so you can update images used in a web app without updating the web app itself.
