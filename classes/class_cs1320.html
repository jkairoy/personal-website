CSCI1320
CS1320 starts with HTML, covers JavaScript in the front end, various back end technologies, databases, as well as security, scaling and testing issues. It takes a holistic look at the process of developing web and mobile applications and aims to bring the students to a point of mastery of many of the most used web technologies and development practices.
Notable Projects
Chatroom
In this project, I implemented a chatroom using two data protocols, SQL, and Socket.io. SQL was used to fetch and store different chatrooms full of data. Socket.io, on the other hand, was used to deliver and recieve immediate communications. The user could view other users in the chatroom, including their AFK status, they could create a new chatroom, change between chatrooms, and change their nicknames.
Final Project
For the final project, myself and two of my classmates created a content management system to help TA's keep track of final project contracts. We achieved this using Jade templating, and a Node.js backend. Through this site, TA's were able to discuss different contract applications, recieve and process applications, and automatically assign students to contracts.
Twitter Clone
In this project, I implemented a simple clone of twitter using pure JavaScript dom manipulation and a python backend connected to the Twitter API.
